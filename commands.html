• git init: to init git in the project
  • git add . : to add file in stage
  • git status: to show the files status
  • git commit -m 'message': to commit the changes
  • git remote add origin https://github.com/rshar29/Gitpractice.git : to add a remote repository in github
  • git branch -M main: to initialize the main branch
  • git push -u origin main: to push the files to main branch
  • git reset --hard f7a7bbc7aa8ac8a511950fa5d1fd5cd333eab0ad : to revert all the new changes and code will be deleted
  • git reset --soft f7a7bbc7aa8ac8a511950fa5d1fd5cd333eab0ad : to revert all the new changes and code will not be deleted
  • git branch: to verify the current branch
  • git branch test : to create a new branch named test
  • git checkout test : to switch to the test branch
  • git merge main : to merge the main branch into test branch
